>To use the system, in the command line the format should be > java Main inputFile.txt outputFile.txt <

>"outputFile" is the output .txt filename the user desires. If a pre-existing file matches the name 
the new output will overwrite the result in the pre-existing file.

>Please ensure to add .txt after the name (eg. inputFile.txt)
